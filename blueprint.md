# Project Blueprint: Modern Portfolio Website

## Overview

This project is a modern, professional portfolio website for Sujin Lee, a software developer and cloud engineer. The site will feature a clean, dark-themed design with smooth animations and interactive elements to create an engaging user experience.

## Design & Style

*   **Theme:** Professional Dark Theme
*   **Primary Color:** Cool, dark blue-gray (`#111827`)
*   **Accent Color:** Vibrant, professional light blue (`#38BDF8`) for interactive elements.
*   **Typography:** "Poppins" font for a clean, modern look.
*   **Animations:**
    *   Subtle fade-in animations on scroll.
    *   Smooth transitions on hover for links and buttons.
    *   Interactive hero section animation.

## Core Sections

1.  **Hero/Welcome Section:** A full-screen introduction with a striking headline and animated background elements.
2.  **About Me:** A brief introduction, a summary of skills, and professional experience.
3.  **Portfolio:** A grid showcasing key projects with images, descriptions, and links.
4.  **Contact:** A simple and elegant contact form.

## File Structure

*   `index.html`: Main HTML file.
*   `style.css`: Main stylesheet for styling.
*   `main.js`: JavaScript for animations and interactivity.

## Current Plan

**Objective:** Build a new, modern portfolio website from scratch.

**Steps:**

1.  **Create `blueprint.md`:** Document the plan for the new website. *(Completed)*
2.  **Create `index.html`:** Build the HTML structure with all the necessary sections and link the CSS and JS files.
3.  **Create `style.css`:** Implement the professional dark theme and add styles for all sections, including basic animations.
